# Delete a Tag

Delete a tag by ID or fully qualified name. Supports soft delete (default), hard delete, and restore operations.

## Delete by ID

<ParamField path="id" type="string" required>
  UUID of the tag to delete.
</ParamField>

<ParamField query="recursive" type="boolean" default="false">
  Recursively delete child tags.
</ParamField>

<ParamField query="hardDelete" type="boolean" default="false">
  Permanently delete the tag. If `false`, the tag is soft-deleted and can be restored.
</ParamField>

## Delete by Name

Use `DELETE /v1/tags/name/{fqn}` to delete by fully qualified name.

<ParamField path="fqn" type="string" required>
  Fully qualified name of the tag (e.g., `Certification.Bronze`).
</ParamField>

<ParamField query="recursive" type="boolean" default="false">
  Recursively delete child tags.
</ParamField>

<ParamField query="hardDelete" type="boolean" default="false">
  Permanently delete the tag.
</ParamField>

## Restore a Soft-Deleted Tag

Use `PUT /v1/tags/restore` to restore a soft-deleted tag.

<ParamField body="id" type="string" required>
  UUID of the soft-deleted tag to restore.
</ParamField>

```python DELETE /v1/tags/{id}
from metadata.sdk import configure
from metadata.sdk.entities import Tags

configure(
    host="https://your-company.open-metadata.org/api",
    jwt_token="your-jwt-token"
)

# Soft delete by ID
Tags.delete("68ec158c-5f5e-4277-8a19-5e9e0cd5294f")

# Hard delete
Tags.delete(
    "68ec158c-5f5e-4277-8a19-5e9e0cd5294f",
    hard_delete=True
)

# Delete by name
Tags.delete_by_name("Certification.Bronze")

# Restore a soft-deleted tag
Tags.restore("68ec158c-5f5e-4277-8a19-5e9e0cd5294f")
```

```java DELETE /v1/tags/{id}
import static org.openmetadata.sdk.fluent.Tags.*;

// Soft delete

Tags.delete("68ec158c-5f5e-4277-8a19-5e9e0cd5294f")

```

```bash DELETE /v1/tags/{id}
# Soft delete by ID
curl -X DELETE "{base_url}/api/v1/tags/68ec158c-5f5e-4277-8a19-5e9e0cd5294f" \
  -H "Authorization: Bearer {access_token}"

# Hard delete
curl -X DELETE "{base_url}/api/v1/tags/68ec158c-5f5e-4277-8a19-5e9e0cd5294f?hardDelete=true" \
  -H "Authorization: Bearer {access_token}"

# Delete by name
curl -X DELETE "{base_url}/api/v1/tags/name/Certification.Bronze" \
  -H "Authorization: Bearer {access_token}"

# Restore soft-deleted tag
curl -X PUT "{base_url}/api/v1/tags/restore" \
  -H "Authorization: Bearer {access_token}" \
  -H "Content-Type: application/json" \
  -d '{"id": "68ec158c-5f5e-4277-8a19-5e9e0cd5294f"}'
```

## Returns

Soft delete returns the tag object with `deleted: true`. Hard delete returns no content (204). Restore returns the restored tag object.

---

## Error Handling

| Code | Error Type | Description |
|------|-----------|-------------|
| `401` | `UNAUTHORIZED` | Invalid or missing authentication token |
| `403` | `FORBIDDEN` | User lacks permission to delete this tag |
| `404` | `NOT_FOUND` | Tag with given ID or FQN does not exist |
