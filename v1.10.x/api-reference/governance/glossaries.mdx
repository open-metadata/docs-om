# Glossaries

A **Glossary** is a collection of business terms and definitions that provides a shared vocabulary across an organization. Glossaries help ensure consistent understanding of business concepts and data semantics.

<Info>
Entity schema follows the [OpenMetadata Standard](https://openmetadatastandards.org/governance/glossaries/).
</Info>

## Fully Qualified Name

The FQN format for a glossary is:

```
glossaryName
```

For example: `BusinessGlossary`
