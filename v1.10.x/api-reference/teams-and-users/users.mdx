# Users

A **User** represents a person or bot account in your organization. Users can be assigned to teams, given roles, and associated with domains for governance purposes.

<Info>
Entity schema follows the [OpenMetadata Standard](https://openmetadatastandards.org/teams-users/user/).
</Info>

## Fully Qualified Name

The FQN format for a user is simply the username:

```
username
```

For example: `aaron_johnson0`
