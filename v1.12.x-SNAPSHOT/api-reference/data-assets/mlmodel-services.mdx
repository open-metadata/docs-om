# ML Model Services

An **ML Model Service** represents a connection to a machine learning platform (e.g., Mlflow, Sklearn, SageMaker). It is the root of the ML model hierarchy and contains ML Models.

<Info>
Entity schema follows the [OpenMetadata Standard](https://openmetadatastandards.org/data-assets/ml/mlmodel-service/).
</Info>

## Entity Hierarchy

ML Model Services are at the top of the hierarchy:

<CodeGroup>
```txt ML Model Service
MlModelService (this page)
└── MlModel
```
</CodeGroup>
